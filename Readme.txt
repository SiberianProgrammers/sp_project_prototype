Для копирования проекта требуется вызвать команду ./cloneProject.sh

Для этого находясь в папке /dxs_project_prototype следует сделать
1) chmod +x hello.sh
2) ./cloneProject.sh

Команде сразу же можно передавать аргумент - имя папки, в которую требуется развернуть проект
К примеру, ./cloneProject.sh myProject

В случае ошибки, проверить где лежит bash командой whereis bash и подставить полученное в cloneProject.sh в первой строке, заменив #!/bin/bash на #!* , где * - результат команды whereis bash.

Далее следовать инструкции.
